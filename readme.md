# PantsuWorld
胖次群的一款自动化的脚本对战类RTS游戏

说白了就是一群程序员在养蛊。

## 流程

程序为一个沙箱，在立体的坐标系中释放几个初始"对象"。

对象可以建造"对象"，武器，和建筑物。

对象也可以开采资源，战斗，自毁等等。

对象建造"对象"必定是需要资源，然后按照神经网络对几个方向（具体还没确定）进行变异生成。

变异的越久，需要的资源越多。

# How to play?
沙箱完成时，将放出SDK/WEB API。届时可以实用自己调用。
